# 通用量表研制教学项目（德尔菲 → 预测试EFA → 正式CFA → 随访预测效度）

> **用途**
> 这是一个可复用的"教学用科研项目"技能卡，用于演示量表（问卷）从0到1的研制与验证全流程。
> **条目内容不限制**（任何主题都适用）。
> 产出包括：教学用合成/模拟数据、统计分析、三线表、图表、论文段落与参考文献体系。

> **边界与合规**
> 本 skill 默认用于**教学与方法演示**，数据为 **synthetic/mock（合成/模拟）**。
> **不要**对真实数据"按结论调数据/篡改数据"以获得想要结果。
> 所有关键结果必须能从"原始行级数据"**复算**（可审计）。

---

## 1）背景知识模块（执行前必须先理解）

### 1.1 量表到底在测什么？
- **构念（Construct）**：你要量化的潜在概念（体验、态度、行为、症状负担、风险、满意度等）。
- **情境（Context）**：何时何地测（门诊当场、出院后、随访1个月等）。
- **用途（Use case）**：筛查/分层管理/质控/科研结局/决策支持。

> 同一构念在不同用途/情境下，条目措辞、维度结构、阈值策略会完全不同。

### 1.2 测量学证据链（COSMIN思路）
常见顺序：
1. **内容效度**：覆盖、相关性、可理解、可回答、可操作
2. **结构效度**：EFA探索；CFA确认（竞争模型）
3. **信度**：α/分半/ω/重测
4. **效度**：已知组、收敛/区分、效标关联、**预测效度**
5. 可选：响应性、测量误差、解释性（阈值/MID）

### 1.3 EFA 与 CFA 的分工
- **EFA**：探索结构 + 删题/合并（开发阶段）
- **CFA**：确认结构 + 比较竞争模型（验证阶段）

### 1.4 预测效度与阈值（把"分数"变成"决策"）
- **ROC** 必须要 **二分类结局**（0/1）。如果随访结局是Likert（1–5），需先定义二分规则，例如：
  - **满意=4–5（1）**，不满意=1–3（0）
- 阈值不是"常数"，应考虑：门诊资源、漏报/误报代价、人群差异、校准与外部验证。

### 1.5 教学合成数据的设计原则
- **不要过完美**：避免载荷全>0.90；避免 α>0.95。
- **保留现实噪声**：允许少量交叉载荷（做"挑战版"更适合课堂讨论）。
- **可审计**：所有结论必须能由原始行级数据复算。

## 2）输入要求（条目内容不限制）

你至少提供这5句话（即使条目尚未定稿也行）：
1) 构念是什么？
2) 测量发生在哪个情境与时间点？
3) 量表用途（筛查/分层/质控/结局/决策）？
4) 预期几维（或"不确定"）？
5) 随访外部结局是什么（满意度/事件/评分）？

可选补充：
- 人口学字段（ID、性别、年龄、组别等）
- 预测协变量（首次/严重度/风险因素等）
- 目标样本量（预测试n、正式n、随访n）

---

## 3）标准研究设计（教学友好）

### 阶段A：条目池 + 德尔菲（内容效度）
- 建立 V1 条目池（覆盖充分，20–40条常见）
- 德尔菲两轮：相关性/重要性/清晰度评分 + 文字意见
- 输出：
  - 原始评分矩阵（可复算）
  - I-CVI、S-CVI/Ave
  - Kendall's W（协调系数）
  - 权威系数Cr（如使用）
  - 条目去向轨迹（保留/合并/删除 + 理由）

### 阶段B：预测试 V2 + 条目分析 + EFA筛题
- V2：中等题量（12–20条）
- 预测试样本：100–200常用
- 必做输出：
  1) 描述统计 + 地板/天花板效应
  2) 条目分析（均值SD、偏度峰度、CITC、删除后α）
  3) 信度（α + 分半；可选ω）
  4) EFA（用于解释"为何删题后更稳/核心题更好"）
  5) "V2→V3删题决策"段落（论文可用）

### 阶段C：正式验证 V3（不做EFA时）+ 信度 + CFA
- V3：短量表（6–12条常见）
- 正式样本：≥300更常见
- 若正式阶段不做EFA：
  - 信度：α + 分半（Spearman–Brown）
  - CFA：确认结构；比较竞争模型（3因子 vs 1因子；可选二阶/双因子）

### 阶段D：随访预测效度（抽样随访）
- 从正式样本便利抽取或分层抽取随访 n≈100（或更大）
- 外部结局可为Likert 1–5（教学常用）
- 分析：
  - Spearman相关：基线总分 vs 随访满意度
  - 有序Logistic：基线+协变量 → Likert结局
  - 二元Logistic + ROC/AUC：基线（或模型概率）→ 二分类结局
  - AUC + Youden截点 + Se/Sp（可选bootstrap CI）

## 4）必交付物（交付清单）

### 数据包（Excel）
- **Delphi_raw.xlsx**：专家两轮原始评分 + 文字意见 + 指标计算
- **Pilot_V2.xlsx**：原始行级数据 + 描述统计/条目分析/信度/EFA
- **Formal_V3.xlsx**：原始行级数据 + α/分半 + CFA拟合/载荷/维度相关
- **Followup_100.xlsx**：随访样本 + 结局 + 相关/回归/ROC
- **Recheck.xlsx**：复核表（从原始数据重算关键结果）

### 表图（Word + PNG）
- 三线表Word（SPSS风格）
- CFA拟合与载荷表（Word）
- ROC图（PNG）：
  - 同图叠加：模型ROC vs 仅总分ROC
  - 星标Youden最佳点 + 箭头标注阈值/Se/Sp
  - 中文字体正常显示（宋体风格）

### 论文文本
- 结果：德尔菲→预测试筛题→正式CFA→随访预测效度
- 讨论：结构含义、工具定位、风险因素解释、阈值落地、局限与展望
- 参考文献：≥35条；正文编号引用；文末附 PubMed/DOI 链接

## 5）执行顺序（推荐）

1) 明确背景（第2部分输入）
2) 起草 V1 条目池
3) 做德尔菲 → 形成 V2 + 条目轨迹
4) 生成/收集预测试数据（教学：合成）
5) 预测试：条目分析+信度+EFA → 决策 V3
6) 生成/收集正式数据（教学：合成）
7) 正式：α+分半+CFA（竞争模型）
8) 抽取随访样本并生成随访结局
9) 预测效度：相关+回归+ROC
10) 生成图表与三线表，撰写结果/讨论
11) 终检：复算关键结果；检查引用编号与链接有效性

---

## 6）质量闸门（必须通过）

### 数据"像真"
- 评分分布合理（除非确有理由）
- 载荷不过分完美；α不过高
- 候选不良条目"略差但不离谱"（例如CITC≈0.20–0.29更真实）
- 可选"挑战版"：存在少量交叉载荷便于课堂讨论

### 可复算/可审计
- AUC、阈值、OR、α、载荷等都能从原始数据复算
- 规则清晰记录（例如：ROC二分规则、阈值选择策略）

### 图文一致
- 图画了什么，正文就写什么
- 若文字给出"仅总分AUC/阈值"，要么同图画出总分ROC，要么在表/附录给出并引用

### 引用规范
- 正文引用序号连续且对应文末参考文献
- 文末每条文献附 PubMed 或 DOI（书籍可用出版社链接）

## 7）可复制模板（建议默认）

### 7.1 ROC二分规则（随访满意度1–5）
- **满意（阳性）=4–5**
- **不满意（阴性）=1–3**
并在方法与结果中解释该规则。

### 7.2 "预测试删题决策"段落模板
- "条目分析提示某些条目CITC较低/删除后α提升有限；EFA中载荷偏弱或跨域不稳定……"
- "剔除后结构更清晰且信度维持可接受……"
- "据此形成终版V3……"

### 7.3 ROC解释模板（教学友好）
- "AUC代表随机抽取一名阳性与一名阴性，模型给阳性更高评分的概率……"
- "Youden指数用于平衡敏感度与特异度选择阈值……"
