# 学术英语表达参考

医学论文写作的专业表达、常见错误和替换建议。

## 一、动词替换（避免口语化）

### 表示"显示/表明"

| 避免使用 | 推荐使用 |
|----------|----------|
| show | demonstrate, indicate, reveal, suggest |
| prove | support, confirm, verify |
| find | observe, identify, detect, note |
| get | obtain, acquire, achieve, yield |
| give | provide, offer, present |

**语境选择**：
- demonstrate: 通过数据/实验证明
- indicate: 数据指向某结论
- reveal: 揭示（之前未知的）
- suggest: 暗示（较弱的结论）

### 表示"增加/减少"

| 避免使用 | 推荐使用 |
|----------|----------|
| go up | increase, rise, elevate, enhance |
| go down | decrease, decline, reduce, diminish |
| get better | improve, ameliorate, enhance |
| get worse | deteriorate, worsen, exacerbate |

### 表示"进行/实施"

| 避免使用 | 推荐使用 |
|----------|----------|
| do | perform, conduct, carry out, execute |
| use | employ, utilize, apply, adopt |
| make | create, generate, produce, develop |
| look at | examine, investigate, analyze, assess |

## 二、形容词/副词替换

### 程度表达

| 避免使用 | 推荐使用 |
|----------|----------|
| very | highly, substantially, considerably |
| a lot of | numerous, substantial, considerable |
| big | large, substantial, significant, marked |
| small | minor, minimal, slight, modest |
| important | significant, crucial, critical, essential |

### 频率表达

| 避免使用 | 推荐使用 |
|----------|----------|
| always | consistently, invariably, universally |
| never | rarely, seldom, infrequently |
| sometimes | occasionally, intermittently |
| usually | typically, generally, commonly |

## 三、避免主观表达

### 主观→客观

```
❌ We believe that...
✓ The findings suggest that...
✓ The data indicate that...

❌ It is obvious that...
✓ It is evident that...
✓ The results clearly demonstrate that...

❌ Interestingly, ...
✓ Notably, ...
✓ Of note, ...

❌ Surprisingly, ...
✓ Unexpectedly, ...
✓ Contrary to expectations, ...

❌ We think...
✓ We hypothesize that...
✓ We speculate that... (在Discussion中)
```

### 避免绝对化

```
❌ This proves that...
✓ This supports the hypothesis that...
✓ This suggests that...

❌ X causes Y (观察性研究)
✓ X is associated with Y
✓ X may contribute to Y

❌ always / never
✓ generally / rarely
✓ in most cases / seldom
```

## 四、常见中式英语纠正

### 句式错误

```
❌ With the development of medical technology, more and more patients...
✓ Advances in medical technology have enabled...
✓ With recent advances in medical technology, an increasing number of patients...

❌ In recent years, ... has attracted more and more attention.
✓ ... has received increasing attention in recent years.
✓ There has been growing interest in...

❌ Play an important role in...
✓ Play a crucial/critical/significant role in... (避免important)
✓ Contribute to...
✓ Are involved in...

❌ Has a significant effect on...
✓ Significantly affects...
✓ Has a pronounced effect on...
```

### 冗余表达

```
❌ period of time → period
❌ in order to → to
❌ due to the fact that → because
❌ at the present time → currently / now
❌ a total of 100 patients → 100 patients
❌ whether or not → whether
❌ the reason is because → the reason is that / because
```

### 介词错误

```
❌ superior/inferior than → superior/inferior to
❌ different with → different from
❌ comply to → comply with
❌ similar with → similar to
❌ compared to (在研究中) → compared with
```

## 五、时态使用指南

### Abstract

| 部分 | 时态 | 示例 |
|------|------|------|
| Background | 现在时 | Diabetes is a major health concern. |
| Objective | 过去时/不定式 | This study aimed to... / We aimed to... |
| Methods | 过去时 | We conducted a cohort study. |
| Results | 过去时 | The mean age was 45 years. |
| Conclusion | 现在时 | These findings suggest that... |

### Introduction

```
现在时为主（陈述已知事实和普遍真理）：
- Hypertension is a major risk factor for cardiovascular disease.
- Previous studies have shown that... (现在完成时)

引用具体研究时可用过去时：
- Smith et al. reported that... (过去时)
```

### Methods

```
全部使用过去时：
- Patients were enrolled between January and December 2023.
- Data were collected using standardized questionnaires.
- Statistical analyses were performed using SPSS.
```

### Results

```
过去时为主：
- The mean age was 45.3 years.
- A significant difference was observed between groups.

图表引用可用现在时：
- Table 1 shows the baseline characteristics.
- Figure 2 demonstrates the survival curves.
```

### Discussion

```
混合使用：
- Our study found that... (过去时-本研究结果)
- This finding suggests that... (现在时-解释意义)
- Previous studies have reported... (现在完成时-文献)
- Further research is needed... (现在时-建议)
```

## 六、常用连接词和过渡句

## 七、特定部分常用表达

### Introduction结尾（研究目的）

```
Therefore, this study aimed to investigate...
The purpose of this study was to evaluate...
In the present study, we sought to determine...
We conducted this study to assess...
The objective of this study was to compare...
```

### Methods开头（研究设计）

```
This was a [prospective/retrospective] [cohort/case-control/cross-sectional] study.
We conducted a randomized controlled trial to...
A systematic review and meta-analysis was performed...
This single-center study was conducted at...
```

### Results开头（样本描述）

```
A total of [N] patients were included in the final analysis.
Of the [N] patients screened, [N] met the inclusion criteria.
The study population consisted of [N] patients with [疾病].
During the study period, [N] patients were enrolled.
```

### Discussion开头（主要发现）

```
In this [study type], we found that...
The main finding of our study was that...
Our results demonstrate that...
The present study provides evidence that...
This study shows that...
```

### Discussion结尾（结论）

```
In conclusion, our findings suggest that...
Taken together, these results indicate that...
In summary, this study demonstrates that...
Our study provides evidence supporting...
```

## 八、数字和单位规范

### 数字使用

```
句首用文字：
✓ Twenty patients were enrolled.
✗ 20 patients were enrolled. (句首)

小于10用文字，≥10用数字（部分期刊规定）：
✓ five patients... 15 patients...

精确数据用数字：
✓ The mean age was 45.3 years.
✓ P = 0.042
```

### 单位使用

```
数字和单位之间空一格：
✓ 45 kg, 120 mmHg, 5 mL
✗ 45kg, 120mmHg, 5mL

百分号紧跟数字：
✓ 45.2%
✗ 45.2 %

温度：
✓ 37°C (无空格)
```

## 九、缩写使用规范

```
首次出现时写全称：
✓ body mass index (BMI)
✓ confidence interval (CI)
✓ standard deviation (SD)

通用缩写无需定义：
DNA, RNA, PCR, CT, MRI, ECG, BMI (部分期刊)

避免在标题和摘要中使用缩写（除非非常通用）

避免句首使用缩写
```
