# 报告规范 Checklist

各类研究的报告规范核心条目。

## CONSORT 2010（随机对照试验）

### Title and Abstract
- [ ] 1a. 标题中标明为随机试验
- [ ] 1b. 结构式摘要包含试验设计、方法、结果、结论

### Introduction
- [ ] 2a. 科学背景和原理解释
- [ ] 2b. 具体目的或假设

### Methods
- [ ] 3a. 试验设计描述（如平行、析因），包括分配比例
- [ ] 3b. 试验开始后方案的重要改变及原因
- [ ] 4a. 受试者入选标准
- [ ] 4b. 数据收集的地点和场所
- [ ] 5. 干预措施详细描述，足以重复
- [ ] 6a. 预先明确的主要和次要结局指标
- [ ] 6b. 结局指标的任何改变及原因
- [ ] 7a. 样本量确定方法
- [ ] 7b. 中期分析和试验终止指南（如适用）
- [ ] 8a. 产生随机分配序列的方法
- [ ] 8b. 随机化类型及限制条件
- [ ] 9. 分配隐藏机制
- [ ] 10. 随机分配的实施者
- [ ] 11a. 盲法实施对象
- [ ] 11b. 干预措施的相似性描述
- [ ] 12a. 统计方法
- [ ] 12b. 亚组分析和调整分析方法

### Results
- [ ] 13a. 各阶段受试者流程图
- [ ] 13b. 排除原因
- [ ] 14a. 入组和随访日期
- [ ] 14b. 试验结束或终止原因
- [ ] 15. 基线人口学和临床特征表
- [ ] 16. 各组分析的受试者数量
- [ ] 17a. 主要和次要结局结果，含效应大小和精确度
- [ ] 17b. 二分类结局建议同时报告绝对和相对效应
- [ ] 18. 所有重要不良事件
- [ ] 19. 试验注册号和注册机构

### Discussion
- [ ] 20. 局限性
- [ ] 21. 可推广性
- [ ] 22. 与其他相关证据的一致性

### Other
- [ ] 23. 注册号
- [ ] 24. 获取完整试验方案的途径
- [ ] 25. 资助来源

---

## STROBE（观察性研究）

适用于队列研究、病例对照研究、横断面研究。

### Title and Abstract
- [ ] 1. 标题或摘要中说明研究设计

### Introduction
- [ ] 2. 背景/原理
- [ ] 3. 目的

### Methods
- [ ] 4. 研究设计：关键要素
- [ ] 5. 场所：地点、日期、随访时间
- [ ] 6. 参与者
  - 队列：入选标准、来源、随访方法
  - 病例对照：病例定义、对照来源、选择理由
  - 横断面：入选标准、来源
- [ ] 7. 变量：结局、暴露、潜在混杂因素
- [ ] 8. 数据来源/测量方法
- [ ] 9. 偏倚：处理潜在偏倚的措施
- [ ] 10. 样本量
- [ ] 11. 定量变量处理方法
- [ ] 12. 统计方法
  - 混杂因素控制
  - 亚组和交互作用
  - 缺失数据处理
  - 病例对照：匹配分析
  - 队列：失访处理

### Results
- [ ] 13. 参与者
  - 各阶段人数
  - 未参与原因
  - 流程图
- [ ] 14. 描述性数据
  - 人口学特征
  - 暴露信息
  - 潜在混杂因素
  - 缺失数据情况
  - 队列：随访时间
- [ ] 15. 结局数据
  - 队列：事件数或汇总指标
  - 病例对照：各类别人数
  - 横断面：事件数或汇总指标
- [ ] 16. 主要结果
  - 未调整估计值
  - 调整估计值及调整的混杂因素
  - 连续变量：有意义的类别
- [ ] 17. 其他分析（亚组、交互、敏感性）

### Discussion
- [ ] 18. 主要结果
- [ ] 19. 局限性
- [ ] 20. 解释
- [ ] 21. 可推广性

### Other
- [ ] 22. 资助来源
