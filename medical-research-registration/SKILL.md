# 医学研究登记备案表格 — 填写辅助工具

## 核心功能

根据用户提供的研究方案等材料，**自动生成多种医学研究相关表格的填写内容**，支持国家系统备案和医院内部备案两种场景。

---

## 支持的表格类型

### 📋 表格类型1：医学研究登记备案信息系统
**适用场景**：需要在国家医学研究登记备案信息系统（medicalresearch.org.cn）注册临床研究项目
**特点**：字段详细，需要中英文双语，包含数据共享声明

### 📋 表格类型2：临床研究项目备案表（医院内部）
**适用场景**：医院伦理审查申请、科室内部备案
**特点**：字段精简，中文填写，侧重研究设计和实施信息

### 📋 表格类型3：知情同意书 - 干预性研究
**适用场景**：药物、医疗器械、医疗技术等干预性临床研究
**特点**：包含风险获益详细说明、随机分组、安慰剂等干预性研究特有内容

### 📋 表格类型4：知情同意书 - 非干预性研究
**适用场景**：观察性研究、数据/标本采集类非干预性研究
**特点**：侧重隐私保护、数据使用说明、标本采集风险

### 📋 表格类型5：知情同意书 - 监护人版
**适用场景**：儿童、认知障碍者等无法自行签署知情同意书的受试者
**特点**：面向监护人撰写，强调儿童权益保护

### 📋 表格类型6：知情同意书 - 未成年人版（≥8岁）
**适用场景**：8岁及以上未成年人参与研究
**特点**：儿童友好语言，配合图示，需未成年人+监护人双签

### 📋 表格类型7：免除/免签知情同意书申请表
**适用场景**：申请免除知情同意或免除知情同意书签字的伦理审查
**特点**：需详细说明免除理由和隐私保护措施

### 📋 表格类型8：初始审查申请表
**适用场景**：向伦理委员会提交研究项目的初始伦理审查申请
**特点**：包含研究者信息、产品分类、多中心信息等

## 工作流程

### 通用工作流程（表格类型1-2, 7-8）

```
[步骤1] 用户选择表格类型
    ↓
[步骤2] 收集材料 → 要求用户提供研究方案/伦理批件/相关材料
    ↓
[步骤3] 解析材料 → 提取研究设计、人群、干预、结局等关键信息
    ↓
[步骤4] 信息补全 → 对材料中缺失的字段向用户提问
    ↓
[步骤5] 生成填写内容 → 按所选表格的字段逐项输出
    ↓
[步骤6] 输出交付 → 生成结构化文档，方便复制填报
```

### 知情同意书生成流程（表格类型3-6）

```
[步骤1] 用户选择知情同意书类型
    ↓
[步骤2] 收集材料 → 研究方案、研究者信息、伦理委员会信息
    ↓
[步骤3] 解析材料 → 提取研究目的、设计、风险获益、纳排标准等
    ↓
[步骤4] 信息补全 → 确认受试者人群特点、特殊风险、联系方式等
    ↓
[步骤5] 生成知情同意书 → 按模板结构生成完整文档
    ↓
[步骤6] 输出交付 → 生成可直接使用的知情同意书文本
```

# 表格类型1：医学研究登记备案信息系统

## 材料收集

**必须提供（至少一项）：**
1. **研究方案**（最重要）— 包含研究设计、目的、方法、纳排标准、样本量等
2. **伦理审批件** — 包含伦理委员会名称、批准时间、批件编号

**建议提供：**
3. **项目负责人信息** — 姓名、电话、邮箱、科室、单位地址
4. **立项文件** — 立项名称、编号、资助来源、经费
5. **知情同意书或豁免申请**

## 材料解析

从用户提供的材料中提取以下关键信息：

| 信息类别 | 需要提取的内容 |
|---------|-------------|
| 研究基本信息 | 中英文题目、研究类型（观察性/干预性）、研究阶段 |
| 研究设计 | 队列/横断面/病例对照/RCT等、前瞻性/回顾性 |
| 研究目的 | 主要目的（预防/诊断/治疗等）、具体描述 |
| 研究人群 | 年龄范围、性别、纳入标准、排除标准、样本量及计算依据 |
| 分组信息 | 各组名称、样本量、暴露/干预因素 |
| 结局指标 | 主要指标、次要指标、测量方法、测量时间点 |
| 伦理信息 | 伦理委员会、批准时间、是否免除知情同意 |
| 实施信息 | 经费来源、研究期限、征募时间、单中心/多中心 |
| 负责人信息 | 姓名、联系方式、单位、科室 |

## 信息补全

对材料中**未能提取到**的字段，向用户逐一确认。采用批量提问方式，一次性列出所有缺失字段。

### 常见需要补充的字段

- 研究负责人手机号、邮箱
- 立项编号、经费金额
- 伦理审批件编号和批准日期
- 研究团队成员名单
- 是否需要在中国临床试验注册中心公开
- 数据共享声明
- 征募研究对象的起止时间

### 可推断/默认填写的字段

以下字段如果材料中未提及，可以基于研究类型合理推断：

| 字段 | 默认推断规则 |
|-----|-----------|
| 是否是诊断试验 | 除非研究明确涉及诊断方法评价，默认"否" |
| 是否以中医理论为指导 | 除非明确涉及中医，默认"否" |
| 是否侵入式脑机接口 | 默认"否" |
| 接受健康志愿者 | 根据纳入标准判断 |
| 是否随机分组 | 观察性研究默认"否" |
| 研究阶段 | 观察性研究默认"不适用" |
| 是否采用额外检查 | 回顾性研究默认"否" |
| 涉及生物样本采集 | 除非方案明确提及，默认"否" |
| 涉及国际合作 | 除非方案明确提及，默认"否" |
| 涉及材料捐献 | 除非方案明确提及，默认"否" |

| 研究实施单位 | 从伦理批件或方案中提取，填写完整单位名称 |
| 研究分类 | 根据研究设计判断：研究者发起的临床研究 - 观察性研究/干预性研究 |
| 是否是诊断试验 | 是/否 |
| 暴露因素（干预措施）是否以中医理论为指导 | 是/否 |
| 是否为侵入式脑机接口用于治疗神经精神疾病的临床研究 | 是/否 |

# 表格类型2：临床研究项目备案表（医院内部）

## 材料收集

**必须提供（至少一项）：**
1. **研究方案** — 包含研究设计、目的、方法、纳排标准、样本量等

**建议提供：**
3. **项目负责人信息** — 姓名、电话、科室
4. **立项/经费信息**
5. **研究团队名单**

## 完整字段清单与填写规则

### 一、基本信息

| 字段名称 | 填写规则 |
|---------|---------|
| 备案编号 | （由医院填写，研究者留空） |
| 临床研究题目 | 从方案标题提取 |
| 预期研究时间 | 格式：YYYY年MM月 --- YYYY年MM月 |

### 六、干预措施

| 字段名称 | 填写规则 |

| 方案版本号及版本日期 | 格式：V1.0 / 2026年01月28日 |
| 知情同意书版本号及版本日期 | 与方案版本对应 |

### 知情同意书主体内容（儿童友好语言）

| 章节 | 填写规则 |
|-----|---------|
| **1. 为什么进行这项研究？** | 用简单语言介绍研究目的；可配合图示 |
| **2. 参加研究之前需要做什么？** | 需要签署同意书；监护人也需签署；达到成人年龄或版本更新时需再次签署 |
| **3. 多少孩子将参与这项研究？** | 研究中心数量；入组人数；竞争入组说明 |
| **4. 参加这项研究预计需要持续多长时间？** | 简单说明研究持续时间 |
| **5. 研究会怎样进行？** | 简单语言描述流程（来医院几次、做什么检查、抽几次血、每次多少）；配合图示 |
| **6. 研究中我该做什么？** | 遵守医生指导；按时服药；记录日记；按时随访；报告不适 |
| **7. 参加研究对我们的日常生活会有什么影响？** | 检查对学习的影响；监护人工作影响；不能再参加其他研究 |
| **8. 参加这项研究我会面临怎样的风险？** | 简单描述可能的不适（如打针会痛、可能头晕等）；告诉孩子感到不适时要告诉父母或医生 |
| **9. 参加本研究受试者可能的获益？** | 简单说明可能的好处或对其他孩子的帮助 |
| **10. 如果不参加此研究，有没有其他备选治疗方案？** | 说明可选择不参加 |
| **11. 是否一定要参加并完成本项研究？** | 强调孩子自己决定；可以随时退出；退出前做安全检查；医生仍会照顾 |
| **12. 如果有问题或困难，该与谁联系？** | 医生姓名、电话；伦理委员会名称、电话、地址 |

### 签字页

| 字段名称 | 填写规则 |
|---------|---------|
| 未成年人签名 | 手写签名（≥8岁） |
| 日期 | 签署日期 |
| 监护人签名 | 手写签名 |
| 日期 | 签署日期 |
| 研究者签名 | 手写签名 |
| 日期 | 签署日期 |

# 表格类型7：免除/免签知情同意书申请表

## 适用场景

## 完整字段清单与填写规则

### 基本信息

| 字段名称 | 填写规则 |

| 主要研究者 | 姓名 |
| 承担科室 | 科室名称 |

### 一、免除知情同意（不适用请跳过）

#### （一）利用已有数据/标本且已有知情同意

| 字段名称 | 填写规则 |
|---------|---------|
| 原知情同意书对应的伦理批件号或已签署的知情同意书 | 提供原批件号或上传知情同意书 |
| 本研究是否超出原知情同意书许可的范围 | 如是，详细说明 |
| 受试者隐私和个人身份信息是否得到保护 | 说明保护措施 |
| 后续是否需要随访或再次向受试者获取信息 | 如是，说明具体情况 |

#### （二）利用已有数据/标本但无知情同意

| 字段名称 | 填写规则 |
|---------|---------|
| 所使用的数据或标本为以往临床诊疗、疾病监测或临床研究中获得的 | 说明数据来源 |
| 是否使用受试者明确拒绝利用的数据或标本 | 说明如何排除拒绝者 |

| 受试者隐私和个人身份信息是否得到保护 | 说明保护措施 |
| 后续是否需要随访或再次向受试者获取信息 | 如是，说明具体情况 |

### 二、申请免除知情同意书签字（不适用请跳过）

| 字段名称 | 填写规则 |
|---------|---------|
| 签了字的知情同意书会对受试者的隐私构成不正当的威胁 | 说明联系受试者真实身份和研究的唯一记录是知情同意文件，主要风险来自身份或隐私泄露 |
| 其它情况（如通过邮件、微信、钉钉或电话等方式进行问卷调研） | 说明已向受试者或监护人提供书面知情同意或获得口头知情同意 |

| 字段名称 | 填写规则 |
|---------|---------|
