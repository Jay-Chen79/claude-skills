# 医学研究登记备案信息系统 — 表格填写辅助工具

## 核心功能

根据用户提供的研究方案等材料，自动生成医学研究登记备案信息系统的全部字段填写内容，输出为结构化表格，可直接用于系统填报。

---

## 工作流程

```
[步骤1] 收集材料 → 要求用户提供研究方案/伦理批件/相关材料
    ↓
[步骤2] 解析材料 → 提取研究设计、人群、干预、结局等关键信息
    ↓
[步骤3] 信息补全 → 对材料中缺失的字段向用户提问
    ↓
[步骤4] 生成填写内容 → 按系统字段逐项输出中英文内容
    ↓
[步骤5] 输出交付 → 生成结构化文档（Excel或Markdown），方便复制填报
```

---

## 步骤1：材料收集

### 启动对话

当用户触发此skill时，Claude应主动要求用户提供以下材料（按重要性排序）：

**必须提供（至少一项）：**
1. **研究方案**（最重要）— 包含研究设计、目的、方法、纳排标准、样本量等
2. **伦理审批件** — 包含伦理委员会名称、批准时间、批件编号

**建议提供：**
3. **项目负责人信息** — 姓名、电话、邮箱、科室、单位地址
4. **立项文件** — 立项名称、编号、资助来源、经费
5. **知情同意书或豁免申请**

### 提问话术示例

```
您好！我来帮您生成医学研究登记备案系统的填写内容。

请提供以下材料（可以直接上传文件或粘贴文字）：

📋 **必须**：研究方案（PDF/Word/文字均可）
📋 **建议**：伦理审批件、负责人联系方式、立项/经费信息

上传后我会逐项生成所有字段的填写内容。如有信息缺失，我会单独向您确认。
```

## 步骤2：材料解析

从用户提供的材料中提取以下关键信息：

| 信息类别 | 需要提取的内容 |
|---------|-------------|
| 研究基本信息 | 中英文题目、研究类型（观察性/干预性）、研究阶段 |
| 研究设计 | 队列/横断面/病例对照/RCT等、前瞻性/回顾性 |
| 研究目的 | 主要目的（预防/诊断/治疗等）、具体描述 |
| 研究人群 | 年龄范围、性别、纳入标准、排除标准、样本量及计算依据 |
| 分组信息 | 各组名称、样本量、暴露/干预因素 |
| 结局指标 | 主要指标、次要指标、测量方法、测量时间点 |
| 伦理信息 | 伦理委员会、批准时间、是否免除知情同意 |
| 实施信息 | 经费来源、研究期限、征募时间、单中心/多中心 |
| 负责人信息 | 姓名、联系方式、单位、科室 |

## 步骤3：信息补全

对材料中**未能提取到**的字段，向用户逐一确认。采用批量提问方式，一次性列出所有缺失字段，避免多轮来回。

### 常见需要补充的字段

- 研究负责人手机号、邮箱
- 立项编号、经费金额
- 伦理审批件编号和批准日期
- 研究团队成员名单
- 是否需要在中国临床试验注册中心公开
- 数据共享声明
- 征募研究对象的起止时间

### 可推断/默认填写的字段

以下字段如果材料中未提及，可以基于研究类型合理推断：

| 字段 | 默认推断规则 |
|-----|-----------|
| 是否是诊断试验 | 除非研究明确涉及诊断方法评价，默认"否" |
| 是否以中医理论为指导 | 除非明确涉及中医，默认"否" |
| 是否侵入式脑机接口 | 默认"否" |
| 接受健康志愿者 | 根据纳入标准判断 |
| 是否随机分组 | 观察性研究默认"否" |
| 研究阶段 | 观察性研究默认"不适用" |
| 是否采用额外检查 | 回顾性研究默认"否" |
| 涉及生物样本采集 | 除非方案明确提及，默认"否" |
| 涉及国际合作 | 除非方案明确提及，默认"否" |
| 涉及材料捐献 | 除非方案明确提及，默认"否" |

## 步骤4：生成填写内容

### 输出格式

按照系统的分区结构，逐字段输出**字段名称**和**填写内容**。对于需要中英文双语的字段，同时提供中英文内容。

### 完整字段清单与填写规则

#### 一、分类信息

| 字段名称 | 填写规则 |
|---------|---------|
| 研究实施单位 | 从伦理批件或方案中提取，填写完整单位名称 |
| 研究分类 | 根据研究设计判断：研究者发起的临床研究 - 观察性研究/干预性研究 |
| 是否是诊断试验 | 是/否 |
| 暴露因素（干预措施）是否以中医理论为指导 | 是/否 |
| 是否为侵入式脑机接口用于治疗神经精神疾病的临床研究 | 是/否 |
| 是否需要在其他研究注册平台公开 | 根据用户要求：中国临床试验注册中心公开/否 |
| 项目填报状态 | 默认"预填报" |

#### 二、基本信息 — 项目信息

| 字段名称 | 填写规则 |
|---------|---------|
| 医学研究题目 | 中文题目，从方案标题提取 |
| Scientific title | 英文题目，从方案提取或翻译生成 |
| 医学研究题目简写 | 可留空或取题目关键词缩写 |
| English Acronym | 可留空或生成英文缩写 |
| 医学研究通俗名称 | 通常与医学研究题目相同 |
| Public title | 通常与Scientific title相同 |

#### 三、实施信息

| 字段名称 | 填写规则 |
|---------|---------|
| 本研究经费来源 | 从立项文件提取 |
| 资助级别 | 国家级/省部级/厅局级/自选课题（自筹）等 |
| Source(s) of funding | 英文翻译 |
| 立项名称 | 从立项文件提取，无则留空 |
| 立项编号 | 从立项文件提取，无则留空 |
| 涉及国际合作 | 是/否 |
| 立项时间 | 格式：YYYY-MM-DD |
| 资金额度 | 单位：万元 |
| 涉及材料捐献 | 是/否 |
| 本研究总经费 | 单位：万元 |
| 研究预计持续时间 | 格式：YYYY-MM-DD - YYYY-MM-DD |
| 征募研究对象时间 | 格式：YYYY-MM-DD - YYYY-MM-DD |
| 伦理委员会 | 完整名称 |
| 伦理委员会批准时间 | 格式：YYYY-MM-DD |
| 伦理审批件 | 文件名（需用户自行上传） |
| 数据监察委员会（DSMB） | 具有/不具有 |
| 研究负责人 | 姓名 |
| 研究负责人联系方式 | 电话号码 |
| 研究负责人科室 | 科室名称 |
| 项目联系人 | 姓名（常与负责人相同） |
| Applicant | 英文姓名 |
| 项目联系人电话 | 电话号码 |
| 项目联系人邮箱 | 邮箱地址 |
| 项目联系人通讯地址 | 中文地址 |
| applicant's address | 英文地址 |
| 项目联系人所在单位 | 中文单位名称 |
| Affiliation of the Registrant | 英文单位名称 |
| 研究团队成员 | 序号、姓名、机构（表格形式） |
| 是否为多中心试验/研究 | 是/否 |

#### 四、研究内容

| 字段名称 | 填写规则 |
|---------|---------|
| 研究状态 | 研究尚未开始/正在进行/已完成 |
| 主要目的 | 预防/诊断/治疗/预后/其他 |
| 研究目的的具体描述 | 从方案提取，1-3句话概括 |
| Objectives of Study | 英文翻译 |
| 研究内容 | 从方案提取，包含设计、方法、分析等，200-500字 |
| 学科分类 | 根据研究领域选择，如"H12 眼科学"下的子分类 |
| 国际疾病分类(ICD-11) | 根据目标疾病选择对应ICD-11编码 |
| 具体疾病或症状 | 中文 |
| Target disease | 英文 |
| 关键词 | 3-5个关键词，空格分隔 |
| 研究类型 | 观察性研究/干预性研究 + 探索性/确证性 |
| 是否采用额外的检查、检验、诊断措施 | 是/否 |
| 研究阶段 | I期/II期/III期/IV期/不适用 |

#### 五、研究设计

| 字段名称 | 填写规则 |
|---------|---------|
| 研究设计 | 队列研究-前瞻性/回顾性、横断面研究、病例对照等 |
| 研究参与者性别 | 男性女性均可/仅男性/仅女性 |
| 研究参与者年龄（最小） | 数字+单位（岁/月/天） |
| 研究参与者年龄（最大） | 数字+单位（岁/月/天） |
| 接受健康志愿者 | 是/否 |
| 涉及弱势群体 | 是/否 |
| 弱势群体类型 | 儿童/孕妇/老年人/残障人士等 |
| 纳入标准 | 逐条列出（中文） |
| Inclusion criteria | 逐条列出（英文） |
| 排除标准 | 逐条列出（中文） |
| Exclusion criteria | 逐条列出（英文） |
| 样本量 | 数字 |
| 样本量计算依据 | 从方案提取计算方法和依据 |
| 是否免除知情同意 | 是/否 |
| 预期第一例研究对象签署知情同意日期 | 日期或"NA" |

#### 六、研究分组信息

每个分组需填写以下字段（重复填写）：

| 字段名称 | 填写规则 |
|---------|---------|
| 分组名称 | 中文组名 |

| 暴露因素类型 | 药物/手术/设备/行为/其他 |
| 暴露因素描述 | 中文描述 |
| 是否随机分组 | 是/否 |
| 统计分析方法描述 | 从方案提取统计方法 |

#### 七、测量指标

| 字段名称 | 填写规则 |
|---------|---------|

#### 八、其他信息

|---------|---------|

| 字段名称 | 填写规则 |

## 步骤5：输出交付

### 输出格式选择

**默认输出**：Markdown表格格式，直接在对话中展示，方便逐项复制。

**可选输出**：如用户要求，生成Excel文件（.xlsx），包含两列：字段名称 | 填写内容。

### Markdown输出模板

```markdown

|-----|---------|
| 研究实施单位 | XXX医院 |
| 研究分类 | 研究者发起的临床研究 - 观察性研究 |
| ... | ... |

## 二、基本信息

| 字段 | 填写内容 |
|-----|---------|
| 医学研究题目 | XXXX |
| Scientific title | XXXX |
| ... | ... |

（以此类推，覆盖全部十个分区）
```

## 英文翻译规则

### 翻译质量要求

1. **学术规范**：使用标准医学英语术语，符合学术论文写作规范
2. **一致性**：同一术语在所有字段中保持翻译一致
3. **简洁性**：英文翻译应简洁准确，避免冗余

### 常用翻译对照

| 中文 | 英文 |
|-----|-----|
| 观察性研究 | Observational study |
| 回顾性队列研究 | Retrospective cohort study |
| 前瞻性队列研究 | Prospective cohort study |
| 横断面研究 | Cross-sectional study |
| 病例对照研究 | Case-control study |
| 随机对照试验 | Randomized controlled trial |
| 纳入标准 | Inclusion criteria |
| 排除标准 | Exclusion criteria |
| 主要指标 | Primary outcome |
| 次要指标 | Secondary outcome |
| 自选课题（自筹） | Self-funded |
| 免除知情同意 | Waiver of informed consent |
| 不适用 | Not applicable |

## ICD-11编码辅助

当需要填写ICD-11编码时，Claude应：

1. 根据研究涉及的疾病/症状，在ICD-11分类体系中查找对应编码
2. 提供完整的层级路径（章 → 大类 → 小类 → 具体编码）
3. 如涉及多个疾病，逐一列出所有相关编码
4. 使用中文版ICD-11标准名称

## 质量检查清单

生成内容后，Claude应自查：

- [ ] 所有字段均已填写或标注"待补充"
- [ ] 中英文字段均已提供

- [ ] 样本量与分组样本量之和一致
- [ ] 研究时间线逻辑合理（立项→伦理→征募→研究）
- [ ] ICD-11编码与研究疾病匹配
- [ ] 统计方法描述与研究设计匹配
- [ ] 测量指标与研究目的对应
- [ ] 英文翻译术语准确规范
- [ ] 所有日期格式统一（YYYY-MM-DD）
