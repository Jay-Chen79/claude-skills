# 科普话题库与术语转化

常见医学科普主题的写作参考和术语通俗化指南。

## 眼科科普话题

### 近视相关

| 专业表达 | 科普转化 |
|----------|----------|
| 屈光不正 | 看东西不清楚 |
| 眼轴增长 | 眼球变长了 |
| 调节功能 | 眼睛的"对焦"能力 |
| 睫状肌 | 眼睛里的"对焦肌肉" |
| 角膜曲率 | 眼睛表面的弧度 |
| 屈光度 | 近视的度数 |
| 散瞳验光 | 放大瞳孔后测度数（更准确）|
| 周边离焦 | 镜片边缘的特殊设计 |
| 眼轴生物测量 | 测量眼球长度 |

**热门话题**：
- 孩子近视了怎么办？
- 近视能治愈吗？
- OK镜安全吗？
- 低浓度阿托品是什么？
- 户外活动真的能防近视？
- 近视手术怎么选？

## 数据表达转化

### 发病率/患病率

```
专业：发病率0.3%
科普：每1000个人里有3个会得这个病

专业：患病率5%
科普：每20个人里就有1个有这个问题

专业：10万人中有50例
科普：一个中等规模的体育场，大约有50个人
```

### 风险比/优势比

```
专业：OR=2.0
科普：风险增加了1倍 / 可能性翻倍

专业：HR=0.5
科普：风险降低了一半

专业：降低风险40%
科普：原本10个人会得病，现在只有6个
```

### 生存率

```
专业：5年生存率85%
科普：100个患者，5年后还有85个健在

专业：中位生存期24个月
科普：一半患者能活超过2年
```

## 科普写作禁忌

### 绝对化表达

```
❌ 避免：
- 一定能治好
- 永远不会复发
- 百分之百安全
- 必须这样做

✅ 改为：
- 大多数情况下可以治愈
- 复发的可能性很低
- 总体安全性较高
- 建议这样做
```

### 过度恐吓

```
❌ 避免：
- 不治就会瞎！
- 这个病非常可怕！
- 你可能已经得了！

✅ 改为：
- 及时治疗可以避免视力受损
- 这个病需要重视，但可以管理
- 如果有以下症状，建议检查
```

### 未验证的说法

```
❌ 避免：
- 某某偏方能治XX
- 多吃XX就能预防
- XX是治病神器

✅ 改为：
- 目前研究显示...
- 合理饮食可能有帮助
- XX是辅助措施之一
```

## 常用科普句式

### 引入知识点

```
"你知道吗？..."
"很多人不知道的是..."
"其实，真相是这样的..."
"关于XX，我们常常有个误区..."
"先来做个小测试..."
```

### 解释原因

```
"这是因为..."
"简单来说..."
"打个比方，就像..."
"用大白话说..."
"举个例子..."
```

### 给出建议

```
"建议可以这样做..."
"几个小技巧..."
"日常生活中，可以..."
"需要注意的是..."
"如果出现以下情况，建议就医..."
```

### 过渡承接

```
"说到这里，你可能会问..."
"接下来我们来看..."
"除此之外..."
"更重要的是..."
"话说回来..."
```

### 总结收束

```
"划重点..."
"简单总结一下..."
"记住这几点就够了..."
"最后提醒大家..."
"希望这篇文章能帮到你..."
```

## 科普文章检查表

### 内容准确性
- [ ] 医学信息有依据
- [ ] 没有绝对化表达
- [ ] 区分了"可能"和"一定"
- [ ] 标注了信息来源
- [ ] 提醒了个体差异

### 可读性
- [ ] 专业术语都解释了
- [ ] 有生活化比喻
- [ ] 段落不超过100字
- [ ] 有小标题分段
- [ ] 通顺流畅

### 实用性
- [ ] 有具体建议
- [ ] 有就医指导
- [ ] 有自我判断方法
- [ ] 实操性强

### 安全性
- [ ] 没有推荐未验证疗法
- [ ] 没有过度恐吓
- [ ] 有免责声明
- [ ] 鼓励咨询专业医生
