# 平台写作风格指南

不同平台的科普文章风格差异和最佳实践。

## 微信公众号

### 基本参数
- 标题：64字符以内（建议15-25字）
- 摘要：120字符以内
- 封面：2.35:1 比例（900x383像素）
- 正文图片：建议宽度900px

### 风格特点
```
✅ 适合：
- 深度科普（1500-3000字）
- 系统性知识讲解
- 图文并茂的长文
- 专业权威的内容

📝 写作技巧：
- 开头3秒定生死，第一句要抓人
- 多用小标题分段（用户扫读习惯）
- 重点文字加粗或变色
- 每300字左右插入配图
- 结尾引导关注/转发
```

### 标题风格
```
【优秀示例】
✓ "眼科医生：这3个护眼方法，比眼药水管用多了"
✓ "孩子近视后，90%家长做错的第一件事"
✓ "关于近视手术，眼科主任想说的大实话"

【避免】
✗ "震惊！XXX竟然会导致失明！"（标题党）
✗ "近视的预防和治疗方法"（太平淡）
✗ "论青少年近视防控的重要性"（太学术）
```

### 正文模板
```
[标题]

[封面图]

[导语：1-2句，建立共鸣或抛出问题]

━━━━━━━━━━━━━━━

01 小标题一

正文内容...

[配图]

正文内容...

━━━━━━━━━━━━━━━

02 小标题二

正文内容...

[信息图/表格]

━━━━━━━━━━━━━━━

03 小标题三

正文内容...

━━━━━━━━━━━━━━━

📌 划重点
- 要点1
- 要点2
- 要点3

━━━━━━━━━━━━━━━

[互动引导/关注提示]
[免责声明]
```

## 小红书

### 基本参数
- 标题：20字以内
- 正文：1000字以内（建议300-600字）
- 封面：3:4 或 1:1 比例
- 图片：最多18张

### 风格特点
```
✅ 适合：
- 快速实用的干货
- 个人经验分享
- 清单式/步骤式内容
- 视觉化信息

📝 写作技巧：
- 标题带emoji
- 开头直接点明价值
- 分点罗列（1️⃣2️⃣3️⃣）
- 口语化表达
- 结尾带话题标签
```

### 标题风格
```
【优秀示例】
✓ "👁️近视党必看！眼科医生の护眼秘籍"
✓ "❗后悔没早知道的护眼方法 亲测有效"
✓ "💡关于近视的5个真相 别再交智商税了"

【结构】
emoji + 核心卖点 + 情绪词/身份标签
```

### 正文模板
```
[封面图]

[标题]

姐妹们/家人们！今天必须给大家分享...

[建立信任：身份/经历]
作为一个xxx/我自己xxx

[核心内容，分点]
1️⃣ 第一点
内容简洁，2-3句话

2️⃣ 第二点
内容简洁

3️⃣ 第三点
内容简洁

[补充/提醒]
⚠️ 注意：...

[互动引导]
有问题评论区问我呀～

———
#话题1 #话题2 #话题3
```

### 图片策略
```
图1：封面（吸引点击）
图2-3：核心要点可视化
图4-5：详细说明/步骤
图6：总结/保存图
```

## 抖音/快手（短视频脚本）

### 基本参数
- 时长：15秒-3分钟（建议1-2分钟）
- 字数：口播约150-200字/分钟

### 风格特点
```
✅ 适合：
- 单一知识点
- 反常识内容
- 快问快答
- 演示/对比

📝 脚本技巧：
- 前3秒必须有钩子
- 短句，有节奏
- 口语化
- 有反转或金句
- 结尾互动
```

### 脚本模板

**知识科普型（60秒）**
```
【钩子 0-3秒】
"你知道吗？每天玩手机超过X小时..."

【引入 3-10秒】
"今天用1分钟告诉你..."

【核心内容 10-45秒】
第一点："首先..."
第二点："其次..."
第三点："最重要的是..."

【总结 45-55秒】
"记住这三点..."

【互动 55-60秒】
"你做到了几点？评论区告诉我"
```

**辟谣型（45秒）**
```
【钩子 0-3秒】
"网上说XXX能治近视？大错特错！"

【错误观点 3-10秒】
"很多人觉得..."

【正确解释 10-35秒】
"其实真相是..."
"从医学角度来说..."

【正确做法 35-42秒】
"正确的方法是..."

【互动 42-45秒】
"转发给身边的人"
```

## 微博

### 基本参数
- 正文：2000字以内（建议300-500字）
- 图片：最多9张

### 风格特点
```
✅ 适合：
- 热点解读
- 快速科普
- 图片九宫格
- 话题参与

📝 写作技巧：
- 开头直接亮观点
- 信息密度高
- 可以更犀利
- 善用话题标签
```

### 正文模板
```
【热点切入】
关于#XXX#，说几点：

1. 要点一
简洁说明

2. 要点二
简洁说明

3. 要点三
简洁说明

总结一句话：...

#话题1# #话题2#

[配图1-9]
```

## 医院宣传栏/手册

### 基本参数
- 单页：A4尺寸
- 折页：常用三折/四折
- 字号：正文不小于12pt

### 风格特点
```
✅ 适合：
- 正式权威的内容
- 流程指导
- 疾病介绍
- 就医指南

📝 设计技巧：
- 层次清晰
- 大标题醒目
- 图文配合
- 重点突出
- 联系方式
```

### 内容结构
