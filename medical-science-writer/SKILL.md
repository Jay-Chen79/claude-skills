# 医学科普文章生成器

生成图文并茂的医学健康科普内容。

## 核心理念

```
科普 ≠ 简化专业内容
科普 = 用读者能理解的方式，传递准确的健康知识

好的科普 = 准确性 + 可读性 + 可视化
```

## Step 3: 文章结构设计

### 标准科普文结构

```
┌─────────────────────────────────────────┐
│ 🎯 标题（吸引点击，不标题党）            │
├─────────────────────────────────────────┤
│ 📍 导语/引子                            │
│   - 场景代入 / 提问 / 数据震撼          │
│   - 建立阅读动机                        │
├─────────────────────────────────────────┤
│ 📖 正文                                 │
│   Part 1: 是什么（定义/现象）           │
│   Part 2: 为什么（原因/机制）           │
│   Part 3: 怎么办（建议/方法）           │
├─────────────────────────────────────────┤
│ 💡 总结/金句                            │
│   - 核心要点回顾                        │
│   - 行动呼吁                            │
├─────────────────────────────────────────┤
│ ⚠️ 免责声明（如需要）                   │
└─────────────────────────────────────────┘
```

### 不同平台的结构调整

**微信公众号**：
- 标题15-25字，可用|分隔
- 导语要短，3秒抓住注意力
- 多用小标题分段
- 重点加粗/变色
- 结尾引导关注

**小红书**：
- 标题带emoji
- 开头要"有用"或"有共鸣"
- 分点罗列（1️⃣2️⃣3️⃣）
- 多用口语化表达
- 结尾带话题标签

**短视频脚本**：
- 前3秒必须有钩子
- 口语化，短句
- 有节奏感
- 结尾有互动引导

## Step 4: 写作规范

### 语言转化原则

```
专业表达 → 科普表达

【医学术语转化】
动脉粥样硬化 → 血管壁上长了"垃圾"，像水管生锈
炎症反应 → 身体的"警报系统"启动了
免疫力 → 身体的"防卫军"
代谢 → 身体把食物变成能量的过程
基因 → 身体里的"说明书"

【数据可视化】
"发病率0.3%" → "每1000个人里有3个"
"5年生存率85%" → "100个患者中，5年后85个还健在"
"降低风险40%" → "原本10个人会得病，现在只有6个"

【复杂概念简化】
使用类比：把复杂机制比作日常事物
使用拟人：让器官"说话"
使用故事：用案例带出知识点
```

### 避免的问题

```
❌ 避免：
- 堆砌专业术语
- 长句子（超过40字）
- 大段纯文字（超过100字无配图/分段）
- 吓唬读者（过度强调危害）
- 绝对化表达（一定、必须、永远）
- 未经证实的偏方/疗法

✅ 坚持：
- 准确第一（宁可不说，不能说错）
- 标注信息来源（权威指南、研究等）
- 建议就医的边界（什么情况该看医生）
- 个体差异提醒（仅供参考，因人而异）
```

## Step 5: 配图生成

### 配图类型

```
1️⃣ 题图/封面图
   - 抓眼球，体现主题
   - 风格与内容匹配
   
2️⃣ 知识插图
   - 解释概念
   - 展示结构/流程
   
3️⃣ 信息图（Infographic）
   - 数据可视化
   - 步骤流程
   - 对比展示
   
4️⃣ 漫画/卡通
   - 轻松话题
   - 儿童科普
   
5️⃣ 流程图/示意图
   - 操作步骤
   - 决策流程
```

## Step 6: 专业内容协商机制

### 触发条件
在文章生成过程中，对于以下涉及专业医疗判断的内容，必须启动协商反馈机制：

```
⚠️ 需要用户确认的内容类型：
1. 🔍 诊断建议或流程
   - 如："如果出现XX症状，需要做YY检查"
   - 涉及诊断标准、检查项目建议等

2. 💊 用药指导
   - 如："建议使用XX药物，剂量YY"
   - 涉及药物选择、剂量、使用方法、注意事项等

3. 🩺 治疗方案
   - 如："推荐XX治疗方法，效果YY%"
   - 涉及手术建议、治疗流程、疗效评估等

4. 📊 风险评估
   - 如："XX行为会增加YY风险"
   - 涉及疾病风险、并发症风险、治疗风险等

5. 👥 特殊人群建议
   - 如："孕妇/儿童/老年人应注意XX"
   - 涉及特定人群的健康建议

6. 🚫 禁忌或注意事项
   - 如："XX人群不宜XX"
   - 涉及禁忌症、慎用情况等
```

## 写作技巧库

### 标题公式

```
【数字型】
"关于XXX，你必须知道的5件事"
"3个信号提醒你：该去查查了"
"90%的人都做错的X个习惯"

【问题型】

【对比型】

【悬念型】
"这个习惯，正在悄悄伤害你的XXX"
"被忽视的XXX，可能比你想象的更重要"

【实用型】
"一文读懂：XXX完全指南"
"XXX自查手册：建议收藏"
"手把手教你：XXX正确方法"
```
