# 效应量选择指南

本文档帮助根据研究领域和设计选择合适的效应量。

---

## 效应量概述

效应量是衡量效果大小的标准化指标，不受样本量影响。

### 常用效应量类型

| 类型 | 适用场景 | 示例 |
|-----|---------|-----|
| 均值差异型 | 组间比较 | Cohen's d, Hedges' g |
| 相关型 | 关联分析 | r, R² |
| 比值型 | 分类结局 | OR, RR, HR |
| 重叠型 | 分布比较 | OVL, 共同语效应量 |

---

## Cohen's d 解释标准

### 一般标准 (Cohen, 1988)

| d 值 | 解释 |
|-----|------|
| 0.2 | 小效应 |
| 0.5 | 中效应 |
| 0.8 | 大效应 |

### 领域特定标准

#### 医学/临床研究
| d 值 | 解释 | 临床意义 |
|-----|------|---------|
| < 0.2 | 可忽略 | 无临床意义 |
| 0.2-0.4 | 小 | 可能有临床意义 |
| 0.4-0.6 | 中 | 有临床意义 |
| > 0.6 | 大 | 明显临床意义 |

#### 教育研究
| d 值 | 解释 |
|-----|------|
| 0.2 | 1年增长的1/5 |
| 0.4 | 1年增长的约一半 |
| 0.8 | 相当于2年进步 |

#### 心理学研究
| d 值 | 解释 |
|-----|------|
| 0.1 | 很小 |
| 0.3 | 小 |
| 0.5 | 中 |
| 0.7 | 较大 |
| 1.0 | 大 |

## 比值比 (OR) 解释

### 一般标准

| OR 值 | 解释 |
|------|------|
| 1.0 | 无关联 |
| 1.2-1.5 | 弱关联 |
| 1.5-2.0 | 中等关联 |
| 2.0-3.0 | 较强关联 |
| > 3.0 | 强关联 |
| < 0.67 | 保护效应（对应OR > 1.5） |

### 流行病学参考值

| 暴露因素类型 | 典型OR范围 |
|------------|-----------|
| 生活方式因素 | 1.2-2.0 |
| 职业暴露 | 1.5-3.0 |
| 遗传因素 | 1.1-1.5 |
| 强危险因素 | > 3.0 |

---

## 风险比 (HR) 解释

### 一般标准

| HR 值 | 解释 |
|------|------|
| 1.0 | 无效应 |
| 1.1-1.3 | 微弱效应 |
| 1.3-1.7 | 弱到中等 |
| 1.7-2.5 | 中等到强 |
| > 2.5 | 强效应 |

### 与中位生存时间的关系

对于指数分布：
```
median_exposed / median_unexposed ≈ 1 / HR
```

例如：
- HR = 2.0 → 暴露组中位生存约为非暴露组的一半
- HR = 0.5 → 暴露组中位生存约为非暴露组的两倍

## 相关系数 (r) 解释

### 一般标准

| r 值 | 解释 |
|----|------|
| 0.1 | 小 |
| 0.3 | 中 |
| 0.5 | 大 |

### 领域特定

#### 心理学
| r 值 | 解释 |
|----|------|
| 0.1-0.2 | 小 |
| 0.2-0.3 | 中小 |
| 0.3-0.4 | 中等 |
| 0.4-0.5 | 较大 |
| > 0.5 | 大 |

#### 医学诊断
| r 值 | 解释 |
|----|------|
| < 0.3 | 相关性差 |
| 0.3-0.5 | 轻度相关 |
| 0.5-0.7 | 中度相关 |
| 0.7-0.9 | 高度相关 |
| > 0.9 | 极高相关 |

## R² (决定系数) 解释

### 一般标准

| R² 值 | 解释 |
|------|------|
| 0.02 | 小效应（≈2%变异解释） |
| 0.13 | 中效应 |
| 0.26 | 大效应 |

### 领域特定

#### 社会科学
- R² = 0.10 通常认为可接受
- R² = 0.25 认为较好
- R² = 0.50 认为非常好

#### 自然科学
- 期望通常更高
- R² > 0.70 才认为模型较好

## AUC 解释

### 诊断性能

| AUC 值 | 解释 |
|-------|------|
| 0.50-0.60 | 无诊断价值 |
| 0.60-0.70 | 差 |
| 0.70-0.80 | 一般 |
| 0.80-0.90 | 良好 |
| 0.90-1.00 | 优秀 |

### 筛查 vs 诊断

| 应用 | 目标AUC |
|-----|--------|
| 筛查测试 | ≥ 0.70 |
| 诊断测试 | ≥ 0.80 |
| 金标准替代 | ≥ 0.90 |

## 效应量选择流程

```
1. 确定研究类型
   ├── 组间比较 → d, Hedges' g
   ├── 相关分析 → r, R²
   ├── 分类结局 → OR, RR
   └── 生存分析 → HR

2. 考虑领域惯例
   └── 参考同领域文献的效应量报告

3. 确定预期效应大小
   ├── 基于前期研究
   ├── 基于临床意义
   └── 基于实际可行性

4. 计算所需样本量
   └── 使用功效分析
```

## 常见换算公式

### d 与 r
```
r = d / √(d² + 4)
d = 2r / √(1 - r²)
```

### d 与 OR
```
d = ln(OR) × √3 / π ≈ ln(OR) × 0.551
OR = exp(d × π / √3) ≈ exp(d × 1.814)
```

### r 与 R²
```
R² = r² (简单回归)
```

### OR 与 RR
```
RR = OR / (1 - p0 + p0 × OR)
```
其中 p0 是对照组的基线风险

## 效应量报告规范

### 必要信息
1. 效应量点估计值
2. 95%置信区间
3. 样本量
4. 统计检验p值

### 示例
```
组间差异具有统计学意义（d = 0.65, 95% CI [0.42, 0.88], 
p < .001），效应量为中到大。
```
