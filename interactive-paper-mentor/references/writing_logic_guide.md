# 写作逻辑指南

帮助Claude在交互过程中解释每句话的目的和逻辑关系。

## 一、Introduction逻辑地图

### 标准漏斗结构

```
                    ┌─────────────────────┐
                    │    全球/宏观背景     │ ← 第1段开头
                    │   （流行病学数据）   │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │   聚焦到特定人群     │ ← 第1段中间
                    │ （儿童/地区/亚组）   │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │   问题的严重后果     │ ← 第1段结尾
                    │  （并发症/负担）     │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │   现有干预措施概述   │ ← 第2段开头
                    │                     │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │   具体干预的现状     │ ← 第2段中间
                    │  （阿托品的研究）    │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │   现有研究的局限     │ ← 第2段结尾
                    │   （研究空白）       │
                    └──────────┬──────────┘
                               │
                    ┌──────────▼──────────┐
                    │    本研究的目的      │ ← 第3段
                    │                     │
                    └─────────────────────┘
```

### 句间逻辑关系

| 逻辑关系 | 常用连接词/方式 | 示例 |
|----------|----------------|------|
| 递进 | Furthermore, Moreover, Additionally | 流行→更严重→后果 |
| 转折 | However, Nevertheless, Despite | 有治疗→但效果有限 |
| 因果 | Therefore, Consequently, Thus | 有危害→需要干预 |
| 聚焦 | Particularly, Especially, Notably | 全球→东亚→儿童 |
| 对比 | In contrast, Compared with, Unlike | A药→B药效果不同 |

### 每句话的目的分类

```
【背景建立类】
- 流行病学数据：建立问题的普遍性
- 临床重要性：建立问题的严重性
- 趋势描述：建立问题的紧迫性

【问题引出类】
- 后果阐述：解释为什么重要
- 机制说明：解释问题的成因
- 人群聚焦：缩小讨论范围

【空白指出类】
- 知识空白：指出不清楚的领域
- 争议呈现：指出不一致的结论
- 局限强调：指出现有研究不足

【目的陈述类】
- 研究目标：说明要解决什么
- 假设提出：说明预期发现
- 方法简述：说明如何研究
```

## 二、Methods逻辑地图

### 标准结构

```
┌─────────────────────────────────────────────────────────────┐
│ Study Design → Participants → Intervention → Outcomes → Stats│
└─────────────────────────────────────────────────────────────┘

每个部分都要回答：WHO, WHAT, WHEN, WHERE, HOW
```

### 各部分的核心问题

| 部分 | 必须回答的问题 |
|------|----------------|
| Study Design | 什么设计？在哪里？什么时间？伦理批准？ |
| Participants | 谁纳入？谁排除？多少人？怎么算的样本量？ |
| Intervention | 具体做了什么？对照组怎么处理？ |
| Outcomes | 主要看什么？怎么定义？怎么测量？ |
| Statistics | 用什么方法？为什么用这个方法？ |

### 统计方法选择逻辑

```
数据类型判断：
├─ 连续变量
│   ├─ 正态分布
│   │   ├─ 两组比较 → t检验
│   │   └─ 多组比较 → ANOVA
│   └─ 非正态分布
│       ├─ 两组比较 → Mann-Whitney U
│       └─ 多组比较 → Kruskal-Wallis
│
├─ 分类变量
│   ├─ 无序分类 → 卡方检验/Fisher精确
│   └─ 有序分类 → 秩和检验
│
└─ 生存数据 → Kaplan-Meier + Log-rank / Cox回归
```

## 三、Results逻辑地图

### 标准呈现顺序

```
1. 研究流程（呼应Methods的Participants）
   ↓
2. 基线特征（证明组间可比）
   ↓
3. 主要结局（最重要的发现）
   ↓
4. 次要结局（补充发现）
   ↓
5. 亚组分析（探索性发现）
   ↓
6. 敏感性分析（稳健性验证）
```

### 结果描述的逻辑

```
每个结果应包含：
1. 是什么（具体数值）
2. 差多少（效应量）
3. 可信吗（P值/CI）
4. 重要吗（临床意义，可选）

示例：
"The mean axial elongation was [是什么:0.25 mm] in the atropine 
group versus [比较:0.45 mm] in the placebo group, representing 
a [差多少:44%] reduction [可信吗:(P < 0.001)]."
```

## 四、Discussion逻辑地图

### 标准结构

```
┌─────────────────────────────────────────────────────────────┐
│                     主要发现（开门见山）                      │
│                           ↓                                 │
│     ┌─────────────────────┴─────────────────────┐          │
│     │                与文献比较                   │          │
│     │   ┌───────────┬───────────┬───────────┐   │          │
│     │   │  发现1    │   发现2   │   发现3   │   │          │
│     │   │ 一致/不一致│ 一致/不一致│ 一致/不一致│   │          │
│     │   └───────────┴───────────┴───────────┘   │          │
│     └─────────────────────┬─────────────────────┘          │
│                           ↓                                 │
│                    可能的机制解释                            │
│                           ↓                                 │
│              ┌───────────┴───────────┐                     │
│              │  优势      │    局限   │                     │
│              └───────────┴───────────┘                     │
│                           ↓                                 │
│                  临床意义/未来方向                           │
│                           ↓                                 │
│                        结论                                 │
└─────────────────────────────────────────────────────────────┘
```

### 与文献比较的逻辑

```
情况1：结果一致
├─ 陈述一致性："Our findings are consistent with..."
├─ 引用支持文献
└─ 可选：解释为什么一致（增强可信度）

情况2：结果不一致
├─ 陈述差异："In contrast to [Author], we found..."
├─ 引用不一致的文献
└─ 必须：解释可能原因
    ├─ 人群差异
    ├─ 方法差异
    ├─ 定义差异
    └─ 时间/地域差异
```

### 局限性的逻辑

```
按研究设计的固有局限排序：

【设计相关】
- 观察性研究无法建立因果
- 回顾性设计存在信息偏倚
- 单中心结果推广性受限

【实施相关】
- 样本量限制统计效力
- 随访时间可能不足
- 缺失数据的影响

【测量相关】
- 结局指标的局限
- 潜在混杂未控制
- 选择偏倚的可能
```

## 五、句间过渡技巧

### 自然过渡

```
【不好的过渡】
句1结束。句2开始新内容。（生硬）

【好的过渡】
句1结束。承接句1的关键词开始句2。（自然）

示例：
"...affecting approximately 30% of the population."
"This burden is particularly concerning among children..."
        ↑
    "This burden"承接上句
```

### 常用过渡模式

| 关系 | 过渡方式 |
|------|----------|
| 递进 | Moreover/Furthermore/In addition/Additionally |
| 转折 | However/Nevertheless/Despite this/In contrast |
| 因果 | Therefore/Thus/Consequently/As a result |
| 举例 | For example/For instance/Specifically |
| 总结 | In summary/Taken together/Overall |
| 时间 | Subsequently/Following this/After X months |

## 六、导师式提问清单

### Introduction阶段

```
开始前：
- "您的研究主要想解决什么问题？"
- "您认为这个问题为什么重要？"

写作中：
- "这句话您想强调什么？"
- "这里需要引用支持，您倾向于哪类数据？"
- "接下来您想从哪个角度深入？"

段落结束：
- "这段逻辑您满意吗？"
- "有没有遗漏的重要背景信息？"
```

### Methods阶段

```
开始前：
- "请简要描述您的研究设计"
- "样本量是如何确定的？"

写作中：
- "这个变量是连续还是分类的？"
- "您用了什么统计方法？为什么选择这个？"
- "伦理批准信息是什么？"

段落结束：
- "方法描述足够详细让别人重复吗？"
```

### Results阶段

```
开始前：
- "您的主要发现是什么？"
- "结果与预期一致吗？"

写作中：
- "这个数据的具体值是多少？"
- "统计检验的P值是？"
- "需要报告置信区间吗？"

段落结束：
- "还有其他重要结果要报告吗？"
```

### Discussion阶段

```
开始前：
- "您最想讨论哪个发现？"
- "与既往研究相比如何？"

写作中：
- "您认为可能的机制是什么？"
- "这个局限性如何影响结论？"
- "对临床实践有什么启示？"

段落结束：
- "讨论是否覆盖了所有主要发现？"
- "结论是否呼应了Introduction的目的？"
```
