# 常见品管圈主题参考数据

各类主题的基线值、改善目标、常见要因参考。

## 护理类主题

### 降低住院患者跌倒发生率

**指标定义**：跌倒发生率 = 跌倒例数 / 同期住院患者日 × 1000‰

**基线参考**：
- 一般病房：0.3-0.8‰
- 老年科/神经内科：0.8-1.5‰
- ICU：0.1-0.3‰

**柏拉图常见要因**：
| 要因 | 占比参考 |
|------|----------|
| 如厕时跌倒 | 25-35% |
| 床边活动时跌倒 | 20-30% |
| 行走时跌倒 | 15-25% |
| 转移时跌倒 | 10-15% |
| 其他 | 5-10% |

**鱼骨图要因示例**：
- 人：评估不准确、宣教不到位、巡视不及时
- 机：床栏故障、呼叫铃位置不当、助行器不足
- 料：防滑垫缺失、标识不清
- 法：评估流程不完善、交接班遗漏
- 环：地面湿滑、光线不足、通道障碍

---

### 提高手卫生依从率

**指标定义**：依从率 = 实际执行手卫生次数 / 应执行手卫生次数 × 100%

**基线参考**：
- 一般水平：55-70%
- 较好水平：70-85%
- 优秀水平：>90%

**五个时刻依从率差异**：
| 时刻 | 典型依从率 |
|------|------------|
| 接触患者前 | 50-65% |
| 清洁/无菌操作前 | 70-85% |
| 体液暴露风险后 | 85-95% |
| 接触患者后 | 65-80% |
| 接触患者周围环境后 | 45-60% |

**鱼骨图要因示例**：
- 人：意识不足、习惯未养成、时间紧迫
- 机：洗手设施不足、速干手消位置不便
- 料：手消剂刺激皮肤、纸巾缺失
- 法：培训不足、监督机制缺失
- 环：工作量大、急诊情况多

---

### 降低压力性损伤发生率

**指标定义**：发生率 = 新发压疮例数 / 同期高危患者数 × 100%

**基线参考**：
- ICU：3-8%
- 普通病房：1-3%
- 改善后目标：<1%

**好发部位分布**：
| 部位 | 占比 |
|------|------|
| 骶尾部 | 35-45% |
| 足跟 | 15-25% |
| 髋部 | 10-15% |
| 肩胛部 | 5-10% |
| 其他 | 10-20% |

---

### 提高护理文书书写合格率

**指标定义**：合格率 = 合格份数 / 抽查总份数 × 100%

**基线参考**：75-88%
**目标参考**：92-98%

**常见缺陷分布**：
| 缺陷类型 | 占比 |
|----------|------|
| 记录不及时 | 25-35% |
| 内容不完整 | 20-30% |
| 签名缺失/不规范 | 15-20% |
| 时间逻辑错误 | 10-15% |
| 涂改不规范 | 5-10% |

## 医疗质量类主题

### 降低标本不合格率

**指标定义**：不合格率 = 不合格标本数 / 送检标本总数 × 100%

**基线参考**：
- 血液标本：2-5%
- 尿液标本：3-8%
- 微生物标本：5-10%

**不合格原因分布**：
| 原因 | 占比 |
|------|------|
| 溶血 | 25-35% |
| 凝血 | 15-25% |
| 量不足 | 15-20% |
| 标识错误 | 10-15% |
| 容器错误 | 5-10% |
| 送检超时 | 5-10% |

---

### 缩短急诊患者滞留时间

**指标定义**：平均滞留时间 = 总滞留时间 / 急诊患者人次

**基线参考**：
- 一般医院：4-8小时
- 大型三甲：6-12小时
- 改善目标：缩短20-40%

**滞留环节分布**：
| 环节 | 耗时占比 |
|------|----------|
| 等待检查 | 25-35% |
| 等待检查结果 | 20-30% |
| 等待会诊 | 15-20% |
| 等待床位 | 15-25% |
| 其他 | 5-10% |

---

### 提高抗菌药物使用合理率

**指标定义**：合理率 = 合理使用病例数 / 抽查病例总数 × 100%

**基线参考**：60-75%
**目标参考**：85-95%

**不合理类型分布**：
| 类型 | 占比 |
|------|------|
| 无指征用药 | 25-35% |
| 选药不当 | 20-25% |
| 用法用量不当 | 15-20% |
| 疗程过长 | 10-15% |
| 联合用药不当 | 5-10% |

## 医技类主题

### 缩短检验报告TAT时间

**指标定义**：TAT = 标本接收到报告发出的时间

**基线参考**：
- 常规生化：2-4小时
- 急诊生化：30-60分钟
- 血常规：20-40分钟

**延迟环节分布**：
| 环节 | 耗时占比 |
|------|----------|
| 标本处理 | 20-30% |
| 仪器等待 | 25-35% |
| 检测分析 | 20-25% |
| 审核发报告 | 15-20% |

---

### 提高影像检查预约准时率

**指标定义**：准时率 = 准时检查人次 / 预约总人次 × 100%

**基线参考**：70-82%
**目标参考**：90-95%

**延迟原因分布**：
| 原因 | 占比 |
|------|------|
| 患者迟到 | 30-40% |
| 上一检查超时 | 20-30% |
| 设备故障 | 10-15% |
| 准备不充分 | 10-15% |
| 急诊插队 | 5-10% |
