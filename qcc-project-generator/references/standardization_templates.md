## 具体SOP示例：住院患者跌倒预防

```
═══════════════════════════════════════════════════════════════
                    标准作业程序（SOP）
═══════════════════════════════════════════════════════════════

文件编号：HL-SOP-2024-001
文件名称：住院患者跌倒风险评估与预防标准作业程序
版本号：V1.0
生效日期：2024年X月X日
制定部门：护理部
审批人：XXX护士长

───────────────────────────────────────────────────────────────
一、目的
───────────────────────────────────────────────────────────────
规范住院患者跌倒风险评估流程，落实预防措施，降低跌倒发生率，
保障患者安全。

───────────────────────────────────────────────────────────────
二、适用范围
───────────────────────────────────────────────────────────────
1. 适用科室：全院各住院病区
2. 适用人员：全体护理人员
3. 适用情境：所有住院患者的跌倒风险管理

───────────────────────────────────────────────────────────────
三、职责分工
───────────────────────────────────────────────────────────────
1. 责任护士：负责跌倒风险评估、宣教、预防措施落实
2. 护士长：负责监督检查、质量控制
3. 护理部：负责培训、考核、持续改进

───────────────────────────────────────────────────────────────
四、操作步骤
───────────────────────────────────────────────────────────────

步骤1：入院评估
  ├─ 操作要点：入院2小时内完成Morse跌倒风险评估
  ├─ 评估内容：跌倒史、步态、意识、用药等6项
  ├─ 注意事项：客观评估，不得漏项
  └─ 时间要求：入院后2小时内

步骤2：风险分级
  ├─ 低风险（0-24分）：常规护理
  ├─ 中风险（25-44分）：加强护理+黄色标识
  ├─ 高风险（≥45分）：重点护理+红色标识+专人看护
  └─ 记录：在护理记录单和床头卡标注

步骤3：预防措施落实
  ├─ 环境：确保床栏上起、地面干燥、呼叫铃在手
  ├─ 宣教：向患者及家属讲解预防要点并签字确认
  ├─ 标识：中高危患者佩戴防跌倒腕带
  └─ 巡视：高危患者每1小时巡视一次

步骤4：动态评估
  ├─ 操作要点：病情变化、用药调整、术后等情况重新评估
  ├─ 频次：每周至少评估1次，病情变化随时评估
  └─ 记录：评估结果记录于护理记录单

步骤5：交接班
  ├─ 内容：高危患者作为重点交接内容
  ├─ 方式：床旁交接，确认预防措施落实情况
  └─ 记录：交接班记录本签名确认

───────────────────────────────────────────────────────────────
五、注意事项
───────────────────────────────────────────────────────────────
1. 评估必须由责任护士本人完成，不得代评
2. 高危患者外出检查需专人陪同
3. 发生跌倒后立即启动应急预案并上报

───────────────────────────────────────────────────────────────
六、相关文件
───────────────────────────────────────────────────────────────
1. 《Morse跌倒风险评估量表》
2. 《跌倒/坠床预防措施告知书》
3. 《跌倒/坠床不良事件上报表》

═══════════════════════════════════════════════════════════════
```
