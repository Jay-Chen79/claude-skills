## 1. AI 写作特征识别清单

以下特征越多，该段落 AI 疑似率越高：

### 结构特征（高权重）
- [ ] "不仅…更是…" / "不仅…同时也…" 并列句
- [ ] "这就好比…" / "就像…一样" 类比（尤其连续多个）
- [ ] "事实上，…" / "实际上，…" 开头
- [ ] "今天，我们就从X角度，为您深入解读Y" 式开场白
- [ ] 每段末尾都有"所以/因此/总之"式总结句
- [ ] 全段只有一个主语（"患者"/"大脑"/"我们"贯穿到底）

### 词汇特征（中权重）
- [ ] "漫长的岁月里" / "在过去漫长的X中"
- [ ] "聪明的大脑" / "强大的可塑性"
- [ ] "如实汇报" / "视觉处理后台"
- [ ] "缓冲期" / "磨合过程"（比喻过于精准工整）
- [ ] "深入解读" / "科学原理" / "令人困惑的场景"

### 节奏特征（中权重）
- [ ] 每句长度高度均匀（均在 25-40 字之间）
- [ ] 逻辑层次过于清晰（"首先…其次…最后"）
- [ ] 无任何语气词（啊、嘛、呢、吧、哦 完全缺失）

## 2. 改写策略：视角注入

**核心手法**：从"第三人称科普说明"改为"第一人称临床叙述"

### 模板 A：护士/医生"我"的经历开场

| 原始 AI 写法 | 改写方向 |
|---|---|
| "在眼科医院的日常工作中，我们经常遇到…" | "做手术室护理这行，我在XX待了好几年了。印象最深的，不是手术本身，反而是…" |
| "许多患者术后第一感觉是…" | "我跟这类患者打交道挺多的。好多人出门第一感觉就是——" |
| "针对这种情况，我们通常建议…" | "我跟这类患者说的第一件事，通常就是：" |

### 模板 B：具体患者场景（"有个大爷/大妈"模式）

```
有位七十多岁的大爷，[手术细节]，[结果好看]，但他[出乎意料的反应]。
跟我说："[患者原话，口语]"
```

加入具体元素：
- 年龄（七十出头、八十多岁）
- 行为细节（皱着眉头、走路踩不稳）
- 直接引语（护士啊，怎么…/跟以前完全不一样）

### 模板 C：结尾改为口语收束

| 原始 AI 写法 | 改写方向 |
|---|---|
| "通常几周到几个月后，聪明的大脑就会自动完成色彩校准" | "等上几周到几个月，大脑自然会把色彩调回来，不会一直这样的" |
| "大脑也会自动通过'修图'功能将其屏蔽" | "大脑自己会把它过滤掉的" |
| "这绝非手术失误，而是…暂时无法处理这种巨大的差异" | "这不是手术出了问题，是大脑还没适应" |

## 3. 改写策略：句式破坏

**核心手法**：打破 AI 标志性的"工整并列+类比+总结"三段式

### 3.1 拆长句为短句群

```
改写前（AI 风格）：
这副天然的"黄滤镜"在过去漫长的岁月里，过滤掉了环境中大部分的蓝光，
大脑为了补偿这种色差，早就在视觉处理后台自动调高了色温，习惯了那种"温馨"的暖色调。

改写后（人类风格）：
这副天然的"黄色滤镜"，过去一直默默工作，挡掉了大部分蓝光。
大脑也早就把这种暖色调当成了正常状态，还以为世界本来就是那个颜色。
```

### 3.2 破折号制造停顿

```
改写前：世界变得冷冰冰的，白墙泛蓝，阳光刺眼。
改写后：第一感觉就是——这世界怎么感觉有点冷，白墙偏蓝，阳光也格外刺眼。
```

### 3.3 去掉"这就好比"类比（AI 最爱用）

```
改写前：这就好比我们给电脑更换了最新的高清显示器，但主机还在运行旧版本的驱动程序。
改写后：（直接说）晶体换了，但大脑里的视觉处理系统还没跟上，得有个磨合期。
```

### 3.4 混合句长

目标节奏：短—短—长—短 交替
```
换进去的人工晶体是透明的。（短）
蓝光一下子就进来了。（短）
视网膜忠实地把新的色彩信息传上去——偏冷、偏蓝。（中）
可大脑里存的那套"白平衡"还是旧晶体年代的，两边一对不上，整个画面就显得过亮、颜色发蓝。（长）
这在医学上叫"蓝视症"。（短）
```

## 4. 改写策略：词汇替换

### 4.1 AI 惯用词 → 人类替代词

| AI 惯用词 | 替换选项 |
|---|---|
| 漫长的岁月 | 几十年、这些年下来 |
| 聪明的大脑 | 大脑（去掉修饰词） |
| 深入解读 | 跟大家讲清楚 |
| 科学原理 | 怎么回事 |
| 视觉处理后台 | 大脑里的处理系统 |
| 缓冲期 | 缓一缓 / 适应一段时间 |
| 令人困惑的场景 | 各种"意外" |
| 事实上，这些并没有意味着 | 其实不是 |
| 针对这种情况，我们通常建议 | 我通常跟他们说 |

### 4.2 加入口语化语气词

适度加入（不过度）：
- 句末：`……的`、`……呢`、`……嘛`
- 感叹：`说实话`、`说起来`、`其实啊`
- 转折：`倒是`、`反而`

## 5. 真实案例对照

来源：2026-02-25 白内障科普文降AI项目（原始AI率 38.7%）

### 案例 1：引言段（96.9% → 预期降至 <25%）

**改写前：**
> 在眼科医院的日常工作中，我们经常遇到这样令人困惑的场景：患者顺利完成了白内障手术，医生查房时视力表检查也恢复到了0.8甚至1.0，但他们脸上却并没有预期的喜悦，反而充满了忧虑……这就好比我们给电脑更换了最新的高清显示器，但主机（也就是我们的大脑）还在运行旧版本的驱动程序，色彩和分辨率的校准自然需要一个重新磨合的过程。今天，我们就从视光护理的角度，为您深入解读这术后三大"怪象"背后的科学原理。

**改写后：**
> 做手术室护理这行，我在温州眼视光待了好几年了。有时候真正让我印象深刻的，不是手术台上那段紧张，反而是术后患者来复查时说的那些话。有位七十多岁的大爷，手术做完第三天复查，视力查到0.9，我们都觉得很不错了。但他皱着眉头问我："护士啊，我怎么看白墙都觉得是蓝色的？跟以前完全不一样。"……不是手术失败，而是大脑还在适应新的"设备"——晶体换了，但视觉处理系统还没跟上，得有个磨合期。今天就借这篇文章，跟大家讲清楚这三种最常见的术后"怪象"。

**改写要点**：第一人称开场、具体患者引语、去掉"这就好比"类比、去掉"深入解读科学原理"套话

---

### 案例 2：说明段（95% → 预期降至 <30%）

**改写前：**
> 许多患者术后第一感觉是世界变得冷冰冰的，白墙泛蓝，阳光刺眼。这其实是因为您刚刚摘掉了一副戴了几十年的"黄色墨镜"……就像一张老照片放久了会发黄一样，这副天然的"黄滤镜"在过去漫长的岁月里，过滤掉了环境中大部分的蓝光。大脑为了补偿这种色差，早就在视觉处理后台自动调高了色温，习惯了那种"温馨"的暖色调。

**改写后：**
> 很多患者术后出门，第一感觉就是——这世界怎么感觉有点冷，白墙偏蓝，阳光也格外刺眼。说起来其实不复杂，就是摘掉的那块旧晶体，几十年下来早被氧化得发黄发棕，颜色深着呢。这副天然的"黄色滤镜"，过去一直默默工作，挡掉了大部分蓝光，大脑也早就把这种暖色调当成了正常状态，还以为世界本来就是那个颜色。

**改写要点**：破折号停顿、去掉"漫长的岁月"、去掉"老照片"类比、缩短句子、加"说起来其实"口语词

---

### 案例 3：建议段（80% → 预期降至 <25%）

**改写前：**
> 针对这种情况，我们通常建议患者在术后前两周外出时坚持佩戴墨镜。这不仅是为了防风沙，更是为了帮眼睛挡掉一部分过强的光线，给大脑一个缓冲期。如果在室内觉得灯光太亮，也可以暂时将白光灯换成暖色调的台灯。通常几周到几个月后，聪明的大脑就会自动完成色彩校准，世界自然会重新变回温暖熟悉的颜色。

**改写后：**
> 我跟这类患者说的第一件事，通常就是：前两周出门戴墨镜，回家把白光灯换成暖光灯，先帮大脑"缓一缓"。也别老是拿以前和现在比，这样只会越比越焦虑。等上几周到几个月，大脑自然会重新把色彩校准调好，颜色就慢慢回来了，不会一直这样下去的。

**改写要点**：去掉"不仅…更是…"、去掉"聪明的大脑"、口语收尾、护士第一人称视角
