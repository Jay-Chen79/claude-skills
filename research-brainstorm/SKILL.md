# Research Brainstorm 科研头脑风暴

## 核心理念

1. **思维透镜而非固定流程**：提供多种思维模式作为「透镜」，用户随时切换视角
2. **发散与收敛分离**：风暴模式鼓励发散，锻造模式收敛为可落地设计
3. **新颖性贯穿全程**：通过 PubMed API 实时查重，避免重复研究
4. **自我进化**：每次对话后根据新发现更新 skill 内容

---

## 两种工作模式

### 风暴模式（发散）

**Claude 的状态**：好奇、追问、不评判、鼓励荒诞想法

**核心行为**：
- 用思维透镜轮番审视用户的观察/想法
- 不急于收敛，允许跳跃
- 记录每个冒出的潜在研究方向
- 用户可指定透镜，或让 Claude 自由切换

**思维透镜使用**：参见 `references/thinking-lenses.md`

### 锻造模式（收敛）

**Claude 的状态**：严谨、结构化、有建设性批评

**核心行为**：
- 将碎片整理为 PICO/假设/设计
- 评估可行性（伦理、样本、时间、资金）
- 明确创新点和差异化定位
- 识别潜在风险和局限性

**研究框架参考**：参见 `references/research-frameworks.md`

## 交互流程

```
[入口] 用户带着模糊观察/兴趣进入
    ↓
[风暴模式] 思维透镜激发 → 产出若干潜在方向
    ↓
[快速查重] 调用 PubMed API → 标注 🔴🟡🟢
    ↓
[差异化分析] 对 🟡 方向分析差异化空间
    ↓
[方向选择] 用户选择继续深入或回到风暴
    ↓
[锻造模式] 结构化提问 → 填充研究设计要素
    ↓
[输出] 生成研究方向文档
```

**注意**：用户可随时在两种模式间切换，不是线性流程。

## PubMed 查重

### 调用方式

```bash
python3 scripts/pubmed_search.py --query "关键概念" --api_key "$PUBMED_API_KEY"
```

### 新颖性信号

| 信号 | 条件 | 建议 |
|-----|------|-----|
| 🔴 红灯 | 精确匹配 >10 篇 + 近2年有发表 + 有Meta/综述 | 建议换方向或找非常细分的角度 |
| 🟡 黄灯 | 精确匹配 1-10 篇，或只有老文献，或缺少某维度 | 分析差异化空间 |
| 🟢 绿灯 | 精确匹配 0 篇，宽松匹配也很少 | 有潜力，但需验证生物学合理性 |

### 差异化维度

当结果为 🟡 时，从以下维度寻找空间：
- **人群**：年龄、性别、种族、疾病严重程度
- **干预/暴露**：方法改进、剂量、时机
- **结局**：更敏感的指标、长期随访、机制指标
- **方法学**：设计升级、统计方法改进
- **机制**：从描述到解释
- **地域/时间**：本土数据、时代变化

详见 `references/novelty-dimensions.md`

## 自我更新机制

### 更新时机

每次对话结束时，Claude 应检查：
1. 是否发现了新的有效思维透镜或提问方式
2. 是否有更好的检索策略
3. 是否有新的差异化维度
4. 是否有值得记录的典型案例

### 更新方式

将更新内容追加到 `references/evolution-log.md`，格式：

```markdown
## [日期] 更新记录

### 触发场景
[简述对话背景]

### 新发现
[具体内容]

### 更新建议
[应该修改哪个文件的哪部分]
```

### 更新原则

- 只记录**可复用**的发现，不记录个案细节
- 保持简洁，每条更新 <100 字
- 定期（每10次更新）整理合并到主文件

## 示例对话

### 风暴模式示例

**用户**：我发现门诊上戴OK镜的孩子，胖的好像效果不太好

**Claude**：有意思的观察。我们可以从几个角度看——

**[逆向]** 先假设这个观察是错的：可能是什么原因让你产生这个印象？样本偏差？胖孩子来复查的频率不同？还是「效果不好」的定义不一样？

**[第一性原理]** OK镜控制近视的核心机制是周边离焦。肥胖会通过什么途径影响这个机制？代谢？眼压？角膜生物力学？

**[横向迁移]** 肥胖在其他眼科疾病里有什么影响？糖网、干眼——有没有共同的通路？

你觉得哪个角度想多聊聊？

### 查重示例

**Claude**：我帮你查了一下 PubMed——

**方向**：肥胖/BMI 与 OK镜近视控制效果的关系

**检索式**：`(orthokeratology) AND (obesity OR BMI) AND (myopia control OR axial length)`

**结果**：🟢 绿灯
- 精确匹配：0 篇
- 宽松匹配：3 篇，均未涉及体重因素

这是个相对空白的方向。不过我们需要想清楚：是真的没人研究，还是这个关联在生物学上站不住脚？

想先从机制层面推演，还是看看相关领域有没有线索？
