# 研究框架参考

用于锻造模式，将发散想法收敛为结构化研究设计。

---

## PICO/PECO 框架

### PICO（干预性研究）

| 要素 | 含义 | 示例 |
|-----|------|-----|
| **P**opulation | 研究人群 | 8-12岁近视儿童 |
| **I**ntervention | 干预措施 | OK镜配戴 |
| **C**omparison | 对照 | 单焦框架眼镜 |
| **O**utcome | 结局指标 | 1年眼轴增长量 |

### PECO（观察性研究）

| 要素 | 含义 | 示例 |
|-----|------|-----|
| **P**opulation | 研究人群 | 成年干眼患者 |
| **E**xposure | 暴露因素 | 每日屏幕使用时间 |
| **C**omparison | 对照 | 低屏幕使用者 |
| **O**utcome | 结局指标 | 泪膜破裂时间 |

## FINER 标准

评估研究问题的质量：

| 标准 | 问题 |
|-----|------|
| **F**easible | 可行吗？样本、时间、资金、技术够吗？ |
| **I**nteresting | 有意思吗？你自己想做吗？ |
| **N**ovel | 新颖吗？有创新点吗？ |
| **E**thical | 伦理吗？能过伦理委员会吗？ |
| **R**elevant | 重要吗？对临床/学科有意义吗？ |

## 研究设计类型

### 按证据等级（从高到低）

1. **系统综述/Meta分析**：综合多个研究
2. **随机对照试验 (RCT)**：干预性研究金标准
3. **队列研究**：前瞻性追踪暴露与结局
4. **病例对照研究**：回顾性比较病例与对照
5. **横断面研究**：某一时点的状态描述
6. **病例报告/系列**：个案描述

### 选择建议

| 研究问题类型 | 适合的设计 |
|------------|-----------|
| 干预效果 | RCT > 队列 > 病例对照 |
| 病因/危险因素 | 队列 > 病例对照 |
| 患病率/现状 | 横断面 |
| 诊断准确性 | 诊断准确性研究 |
| 预后/自然史 | 队列研究 |
| 罕见病/新现象 | 病例报告/系列 |

## 变量类型

| 类型 | 定义 | 处理方式 |
|-----|------|---------|
| 自变量/暴露 | 你想研究的因素 | 明确测量方法 |
| 因变量/结局 | 你想观察的结果 | 选择敏感、可靠的指标 |
| 混杂变量 | 同时影响暴露和结局的因素 | 匹配、分层、多变量调整 |
| 中介变量 | 暴露→中介→结局 | 中介分析 |
| 效应修饰因子 | 在不同亚组中效应不同 | 分层分析、交互作用 |

## 样本量考量

### 需要的信息

1. **效应量**：你期望看到多大的差异？
2. **变异度**：结局指标的标准差是多少？
3. **α 水平**：通常 0.05
4. **检验效能**：通常 0.80 或 0.90
5. **脱落率**：预计流失多少？

### 粗略估算原则

- 主要结局每组至少 30 例才有统计意义
- 多变量分析每个变量至少 10-20 个事件
- 亚组分析需要更大样本

## 可行性评估清单

### 样本可及性
- [ ] 目标人群在哪里？
- [ ] 能招募到足够的样本吗？
- [ ] 招募周期多长？
- [ ] 有没有竞争性研究在抢同一人群？

### 伦理考量
- [ ] 需要知情同意吗？
- [ ] 有没有额外风险？
- [ ] 涉及弱势群体吗（儿童、孕妇等）？
- [ ] 能过伦理委员会吗？

### 资源需求
- [ ] 需要什么设备/检查？
- [ ] 需要多少人力？
- [ ] 预算是多少？
- [ ] 有没有合作者/技术支持？

### 时间线
- [ ] 数据收集需要多久？
- [ ] 随访期多长？
- [ ] 什么时候能出结果？
- [ ] 有没有时间压力（毕业、职称等）？

## 创新点提炼

### 常见创新类型

1. **人群创新**：首次在某人群中研究
2. **方法创新**：使用新技术/新指标
3. **机制创新**：揭示新的作用通路
4. **理论创新**：挑战或修正现有理论
5. **应用创新**：将已知知识用于新场景

### 撰写模板

> 本研究的创新点在于：
> 1. 首次在 [人群] 中探讨 [问题]
> 2. 采用 [新方法] 测量 [指标]
> 3. 从 [新角度] 解释 [现象] 的机制
